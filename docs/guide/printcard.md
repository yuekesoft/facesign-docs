# 胸卡打印设置

1. 点击`系统设置`按钮，选择系统设置界面；

![image-20221216141455044](https://vuepressdocs.oss-cn-hangzhou.aliyuncs.com/docsimages/202212161414110.png)

2. 勾选`签到时是否打印小票`，选择打印机型号，点击`小票打印格式设置`；

![image-20221216141617463](https://vuepressdocs.oss-cn-hangzhou.aliyuncs.com/docsimages/202212161416500.png)

3. 点击`设置胸卡打印格式`，打开格式设置界面。

![image-20221216141822251](https://vuepressdocs.oss-cn-hangzhou.aliyuncs.com/docsimages/202212161418291.png)

### 打印设置变量表如下：

| 格式变量             | 人员信息字段 |
| -------------------- | ------------ |
| [EmpNo]    | 人员编号      |
| [Name]     | 姓名         |
| [Company]  | 单位名称     |
| [Job]      | 职务         |
| [Remark]   | 备注         |

::: tip 提示

双击打印格式设置白色区域，可以设置打印纸张大小

:::
